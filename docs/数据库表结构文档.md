# 数据库表结构文档（用于导入数据）

## 数据库概览

- **数据库:** PostgreSQL
- **名称:** `ENGLISH_1`（配置在 `config.py`）
- **扩展:** 需 `pgvector`（先执行 `CREATE EXTENSION IF NOT EXISTS vector;`）
- **初始化脚本:** `init_db.py`

---

## 核心表：`questions`（题目表）

这是导入数据最关键的表。它使用**父子树形结构**：

```
父题（parent_id IS NULL）
  ├── 文章段落（仔细阅读/选词填空/长篇阅读的文章）
  ├── 写作题目
  └── 翻译题目

子题（parent_id 指向父题）
  ├── 仔细阅读子题
  ├── 选词填空空位
  └── 长篇阅读子题
```

### 字段说明

| 字段 | 类型 | 必填 | 说明 |
|:---|:---|:---:|:---|
| `id` | `BIGSERIAL` | 自动 | 主键，导入时不填 |
| `parent_id` | `BIGINT` | 可选 | 父子关联：子题填父题 id，父题留 NULL |
| `subject_id` | `INTEGER` | **是** | 科目 ID，通常是 1 |
| `type` | `VARCHAR(32)` | **是** | 题型代码（见下方字典表） |
| `title` | `TEXT` | 可选 | 标题或文章标题 |
| `content` | `TEXT` | **是** | 题目正文 / 文章内容 |
| `options` | `JSONB` | 可选 | 选项，格式：`{"A":"...","B":"...","C":"...","D":"..."}` |
| `answer` | `TEXT` | 可选 | 正确答案（如 `"A"`）或参考范文 |
| `explanation` | `TEXT` | 可选 | 答案解析 |
| `difficulty` | `SMALLINT` | 可选 | 难度 1-5 |
| `score` | `FLOAT` | 可选 | 分值（默认 0） |
| `order_index` | `INT` | 可选 | 子题排序（同一父题下从 0 开始） |
| `embedding` | `VECTOR(1024)` | 可选 | pgvector 向量（导入时通常不填） |
| `created_at` | `TIMESTAMPTZ` | 自动 | 默认 `NOW()` |
| `updated_at` | `TIMESTAMPTZ` | 自动 | 默认 `NOW()` |

### `type` 字段取值

| 值 | 含义 | 可有父题 | 可有子题 | 有 options |
|:---|:---|:---:|:---:|:---:|
| `writing` | 写作 | ✗ 独立题 | ✗ | ✗ |
| `translation` | 翻译 | ✗ 独立题 | ✗ | ✗ |
| `careful_reading_passage` | 仔细阅读文章 | ✗ 父题 | ✓ | ✗ |
| `careful_reading_question` | 仔细阅读子题 | ✓ | ✗ | ✓ |
| `banked_cloze_passage` | 选词填空文章 | ✗ 父题 | ✓ | ✗ |
| `banked_cloze_blank` | 选词填空空位 | ✓ | ✗ | ✗ |
| `long_reading_passage` | 长篇阅读文章 | ✗ 父题 | ✓ | ✗ |
| `long_reading_question` | 长篇阅读子题 | ✓ | ✗ | ✓ |

## 导入数据示例

### 独立题（写作/翻译）

```sql
INSERT INTO questions (subject_id, type, content, answer, score) VALUES
(1, 'writing', 'Directions: For this part, you are allowed 30 minutes to write an essay...', '参考范文内容...', 15);

INSERT INTO questions (subject_id, type, content, answer, explanation) VALUES
(1, 'translation', '中国有着悠久的历史和灿烂的文化...', '参考译文内容...', '评分要点说明');
```

### 复合题（仔细阅读）

```sql
-- 先插入文章（父题）
INSERT INTO questions (subject_id, type, title, content)
VALUES (1, 'careful_reading_passage', 'Passage 1', '文章正文内容...')
RETURNING id;
-- 假设得到 id = 10

-- 再插入子题（用 parent_id 关联）
INSERT INTO questions (subject_id, type, parent_id, content, options, answer, order_index, score) VALUES
(1, 'careful_reading_question', 10, 'What is the main idea of the passage?',
 '{"A": "Option A text", "B": "Option B text", "C": "Option C text", "D": "Option D text"}',
 'B', 0, 2);

INSERT INTO questions (subject_id, type, parent_id, content, options, answer, order_index, score) VALUES
(1, 'careful_reading_question', 10, 'According to the passage, which of the following is true?',
 '{"A": "Option A text", "B": "Option B text", "C": "Option C text", "D": "Option D text"}',
 'C', 1, 2);
```

### 复合题（选词填空）

```sql
INSERT INTO questions (subject_id, type, title, content)
VALUES (1, 'banked_cloze_passage', '选词填空', '文章中有十个空需要填入合适的词...')
RETURNING id;
-- 假设 id = 20

INSERT INTO questions (subject_id, type, parent_id, content, answer, order_index) VALUES
(1, 'banked_cloze_blank', 20, '第一个空的题干', 'correct_word', 0),
(1, 'banked_cloze_blank', 20, '第二个空的题干', 'another_word', 1);
```

### 复合题（长篇阅读）

```sql
INSERT INTO questions (subject_id, type, title, content)
VALUES (1, 'long_reading_passage', '长篇阅读文章', '文章正文内容...')
RETURNING id;
-- 假设 id = 30

INSERT INTO questions (subject_id, type, parent_id, content, options, answer, order_index) VALUES
(1, 'long_reading_question', 30, 'Match the following statements with the correct paragraph.',
 '{"A": "Paragraph 1", "B": "Paragraph 2", "C": "Paragraph 3", "D": "Paragraph 4"}',
 'A', 0);
```

## 完整表结构（DDL 参考）

### `questions` 表

```sql
CREATE TABLE questions (
    id              BIGSERIAL PRIMARY KEY,
    parent_id       BIGINT REFERENCES questions(id) ON DELETE CASCADE,
    subject_id      INTEGER REFERENCES subjects(id),
    type            VARCHAR(32) NOT NULL REFERENCES dict_question_types(code),
    title           TEXT,
    content         TEXT NOT NULL,
    options         JSONB,
    answer          TEXT,
    explanation     TEXT,
    difficulty      SMALLINT CHECK (difficulty BETWEEN 1 AND 5),
    score           FLOAT,
    order_index     INT,
    embedding       VECTOR(1024),
    created_at      TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    updated_at      TIMESTAMPTZ NOT NULL DEFAULT NOW()
);

CREATE INDEX idx_questions_parent_id ON questions(parent_id);
CREATE INDEX idx_questions_type ON questions(type);
CREATE INDEX idx_questions_difficulty ON questions(difficulty);
CREATE INDEX idx_questions_subject ON questions(subject_id);
CREATE INDEX idx_questions_created_at ON questions(created_at);
```

## 快速导入指南

1. 先插入 `subjects` 和 `dict_question_types` 字典数据
2. 再插入 `knowledge_points`
3. 插入**父题**（文章/独立题），用 `RETURNING id` 获取生成的 id
4. 用父题 id 插入**子题**，注意 `order_index` 从 0 开始
5. 最后插入 `question_knowledge` 建立题-知识点关联

### 常见错误

- 子题的 `type` 写成了父题类型（如把子题写成 `careful_reading_passage`）
- `options` 格式不是合法 JSON 对象（必须用单引号包裹的 JSON 字符串）
- 复合题中子题的 `parent_id` 指向了不存在的父题 id
- 选词填空子题 `banked_cloze_blank` 不需要 `options` 字段
- 忘记设置 `order_index` 导致子题顺序随机
